# NCAA Basketball Analytics Dashboard

Interactive visualization dashboard analyzing whether offensive or defensive efficiency matters more for winning NCAA Division I men's basketball games.

## Research Question
Does offensive or defensive efficiency contribute more to winning games in NCAA Division I men's basketball?

## Key Findings
- **Offense matters more**: Linear regression coefficient of 0.64 vs 0.45 for defense
- **Statistical significance**: Permutation test with p-value = 0.0005
- **Practical impact**: Offense-heavy teams win 2.6% more games on average
- **Model performance**: R² = 0.60, explaining 60% of win percentage variance

- **Source**: barttorvik.com
- **Coverage**: 2013-2023 (excluding 2020)

## Step-by-Step Deployment Instructions

### Option 1: Deploy via GitHub (Recommended)

1. **Create the project structure locally:**
   ```bash
   mkdir ncaa-basketball-analytics
   cd ncaa-basketball-analytics
   ```

2. **Create all files listed above** with their respective content

3. **Initialize Git and push to GitHub:**
   ```bash
   git init
   git add .
   git commit -m "Initial commit: NCAA Basketball Analytics"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/ncaa-basketball-analytics.git
   git push -u origin main
   ```

4. **Deploy on Vercel:**
   - Go to [vercel.com](https://vercel.com)
   - Click "Add New Project"
   - Import your GitHub repository
   - Vercel will auto-detect React settings:
     - Framework Preset: Create React App
     - Build Command: `npm run build`
     - Output Directory: `build`
   - Click "Deploy"

### Option 2: Quick Deploy via Vercel CLI

```bash
# Install Vercel CLI
npm install -g vercel

# Navigate to project directory
cd ncaa-basketball-analytics

# Deploy
vercel

# For production
vercel --prod
```

## Troubleshooting

**Issue**: Blank page after deploy
- **Fix**: Check build logs in Vercel dashboard
- **Common cause**: Build errors or missing dependencies

**Issue**: Charts not displaying
- **Fix**: Ensure recharts is properly installed
- **Check**: `npm install recharts`

**Issue**: Styling looks broken
- **Fix**: Verify Tailwind CDN is loaded in index.html
- **Alternative**: Install Tailwind as a dependency
